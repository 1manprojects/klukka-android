# Klukka Android Client

This Application requires a running Klukka TimeTracking Server to function.

### AI-Notice

This Android Application was generated mostly with Claude-Code using Sonet 4.5 and Sonet 4.6. It was reviewed and tested by a human.

## Setup

To log in, you first need to create an API token on the Profile page of your Klukka instance. This token will be used for the login credentials in the app.
